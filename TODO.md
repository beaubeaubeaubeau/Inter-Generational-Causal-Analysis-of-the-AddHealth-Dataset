## A. Research pipeline — Task 16 and beyond

The following are the analytic next steps called out in [reference/research_journal.md](reference/research_journal.md#outstanding-uncertainties) and [reference/addhealth_synthesis.md §5.6](reference/addhealth_synthesis.md#56-identification-assumptions-and-target-estimand). Ordered roughly by sequencing — later items assume earlier items are done.

- **[P0] Front-door / IV decomposition for AHPVT.** The load-bearing identification ambiguity (confounder vs. mediator) is unresolved. Plan: (i) fit the three-equation front-door model for `IDGX2 → AHPVT → W4_COG_COMP` with L0+L1 confounders on each arc; (ii) compare the mediation-implied total effect to the covariate-adjusted β; (iii) if the two bracket zero, report bounds rather than a point estimate. Ref: [research_journal.md uncertainty #1](reference/research_journal.md#outstanding-uncertainties).
- **[P0] Clean negative-control battery.** `HEIGHT_IN` is contaminated (adolescent height → peer popularity). Candidates to pilot at W1: blood type (`H4BL0–H4BL5` if in public-use; verify), age at menarche (`H1BC5` or W1 pubertal development), hand-dominance, residential stability pre-W1. For each: (i) confirm public-use availability; (ii) argue non-causality vs. all 24 exposures in a short DAG note; (iii) add as a new D2 variant in task14. Ref: [synthesis §9 D2 callout](reference/addhealth_synthesis.md#9-outcome-battery-primary--multi-outcome-extension).
- **[P0] IPAW for W5 outcomes.** Implement `fit_ipaw(design_frame, outcome_frame, covariates)` that returns stabilised inverse-probability-of-attrition weights; layer on top of `GSW5`. Required before any W5 point estimate for `H5EC1`, `H5ID1`, `H5MN*`, `H5LM5`. Trim at 95th percentile, document stabilised effective N. Ref: [synthesis §9 weight caveat](reference/addhealth_synthesis.md#9-outcome-battery-primary--multi-outcome-extension), [synthesis §5.6](reference/addhealth_synthesis.md#56-identification-assumptions-and-target-estimand).
- **[P1] Per-outcome adjustment sets.** Task 15 used L0+L1+AHPVT uniformly across 12 non-cognitive outcomes for comparability. For Task 16 formal estimation, each outcome needs its own DAG: e.g. `H4BMI` likely needs W1 self-reported weight / BMI in L1; `H5EC1` should drop AHPVT and condition on parental education / cognitive achievement only. Ref: [research_journal.md uncertainty #7](reference/research_journal.md#outstanding-uncertainties).
- **[P1] Sibling-dissociation pairs for non-cognitive outcomes.** D3 is currently hard-coded to cognition-relevant pairs (e.g. `ODGX2` as sibling for peer-network exposures). Handoff pair `IDGX2 → H4WAIST` has no D3 check at all. Extend `SIBLINGS` in [scripts/task14_causal_screening.py](scripts/task14_causal_screening.py) with per-outcome overrides; re-run task15 with D3 active. Ref: [research_journal.md uncertainty #5](reference/research_journal.md#outstanding-uncertainties).
- **[P1] Saturated-school stratification / sensitivity.** `D8` is currently informational only. Quantify the externa-validity gap between saturated and non-saturated schools on L0+L1 covariates; consider saturation-strata fixed effects or a restricted-to-saturated sub-analysis as a sensitivity variant. Ref: [research_journal.md uncertainty #6](reference/research_journal.md#outstanding-uncertainties), [synthesis §5.6 positivity discussion](reference/addhealth_synthesis.md#56-identification-assumptions-and-target-estimand).
- **[P1] E-value / VanderWeele sensitivity bounds on the 4 handoff pairs.** Report the minimum unmeasured-confounder strength (on risk-ratio and correlation scales) required to explain away each of the four Task-16 handoffs. Complements the formal adjustment strategy. Ref: reviewer-A critique (punch-list item: "no sensitivity analysis *strategy*").
- **[P1] Ordered-logit / interval-regression for `H5EC1`, `H5LM5`, `H4BMICLS`.** The screen treats these linearly; formal estimation must not. For `H5EC1` (13 income brackets) use interval regression on bracket midpoints; for `H5LM5` (3-level) multinomial or ordered logit; for `H4BMICLS` (6-level ordinal) ordered logit.
- **[P1] Bring `H4TO*` medication flags into the L1 covariate set for `H4SBP` / `H4DBP`.** Anti-hypertensive use is not currently adjusted; the null D4 on blood-pressure outcomes could be mechanical. Audit which `H4TO*` items are public-use and test sensitivity.
- **[P2] W6 self-reported memory (`H6DA18B`–`H6DA18F`) as a tertiary cognitive outcome.** Public-use W6 has no performance cognition but does have self-reported memory items; these are subjective but add longitudinal signal. Low priority because construct validity is weaker than the W4 battery. Ref: [synthesis §4.6](reference/addhealth_synthesis.md).
- **[P2] Manski bounds as a fallback when IPAW is not credible.** When the attrition-probability model has large variance or sparse support in some strata, Manski-style bounds on the outcome effect (best-case / worst-case imputations for attritted cells) are more honest than a single IPAW estimate. Keep in the back pocket for handoff pairs whose IPAW fit is noisy.
